Планирование движения -  важнейшая задачей в навигации мобильных роботов в пространстве. Задача включает в себя три аспекта.
	Спланированный путь должен проходить из начальной к конечной точке
	Путь должен обеспечивать беспрепятственное перемещение мобильного робота в пространстве.
	Путь должен среди всех возможных путей, являться в определённом смысле оптимальным.
Преимущественно, как следует из литературы,	 чаще всего, на практике применяются методы, предусматривающие решение задачи локального планирования.
 Локальная навигация занимается решением следующей группы задач: определение координат робота по отношению к некоторой точке, локализация робота в рабочем пространстве, 
планирование его маршрута перемещения к цели с предотвращением столкновений с препятствиями в процессе движения. 

Существуют различные способы построить траекторию движения робота. Одним из широко распространённых в локальной навигации мобильного робота является метод потенциальных полей. 
Данный метод позволяет при известных точках препятствий и цели, а также окрестностей рассматриваемой точки совершить следующей шаг в построении траектории.

Метод искусственных потенциалов основан на введении в рассмотрение некоторого поля «информационных сил», 
«воздействие» которого на мобильный робот характеризуется «притяжением» к целевой точке и «отталкиванием» от препятствий.

Данный проект представляет собой реализацию МИПП(метода искусственных потенциальных полей или APF)
Представлен скрипт и файл .exe, являющимся лаунчером для запуска алгоритма.
